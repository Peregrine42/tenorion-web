# MusicGrid (Web-based Tenori-on Clone)

https://raw.github.com/Peregrine42/tenorion-web/main/docs/demo.mp4

## Local Dev Setup

To start the server:
```
bundle install
rails s
```

To build (and watch) the frontend JS assets:
```
npm install
npm run build -- --watch
```
